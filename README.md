## Descrição

O presente programa refere-se à um desafio proposto pela empresa ONFLY em seu processo seletivo. Trata-se de um CRUD backend em nest.js protegido por autenticação de login.



## Instalação

```bash
$ npm install
```

## Executando o App

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Testes

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
